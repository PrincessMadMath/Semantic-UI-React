Divider

A breadcrumb can contain a divider to show the relationship between sections, this can be formatted as text.

You can do the same using shorthands.

Divider as icon

It can be also formatted as icon.

Section

A breadcrumb can contain sections that can either be formatted as a link or text.

Link

A section may be linkable or contain a link.
